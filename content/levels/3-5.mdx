你的网站现在功能齐全了，但图片素材还是用的占位图。这一关教你用 AI 生成专属的图片素材，让网站看起来像花了钱请设计师做的。

## 为什么需要 AI 生图？

传统做法：花几百块买图库会员，或者花几千块请设计师。AI 时代的做法：用自然语言描述你想要的图片，几秒钟就能生成。

## 操作步骤

### 第一步：了解 AI 生图工具

目前主流的免费/低成本 AI 生图工具：

| 工具 | 特点 | 适合场景 |
|------|------|----------|
| **Ideogram** | 免费额度，文字渲染强 | Logo、Banner、带文字的图 |
| **Leonardo.ai** | 免费额度，风格多 | 插画、产品图、背景 |
| **Playground AI** | 完全免费 | 快速出图、迭代 |
| **Stable Diffusion (本地)** | 免费无限制 | 批量生成、隐私要求高 |
| **美境 (aidesign.meituan.com)** | 美团内部 | 商业素材 |

### 第二步：为网站生成 Hero 背景图

对 AI 生图工具说：

```text
A modern abstract gradient background for a tech website, 
dark theme with subtle purple and blue glowing orbs, 
minimal and clean, 16:9 aspect ratio, high quality
```

### 第三步：生成项目展示的 Mockup 图

```text
A clean laptop mockup showing a modern dashboard interface, 
dark mode UI with charts and data visualization, 
isometric view, soft lighting, professional product photography style
```

### 第四步：生成用户头像/团队照片

```text
A friendly professional avatar illustration, 
minimal flat design style, soft pastel colors, 
circular crop, suitable for a tech product
```

### 第五步：把生成的图片集成到项目中

对你的 AI 编程工具说：

```text
帮我把这些 AI 生成的图片集成到项目中：

1. 把 hero-bg.png 设为首页 Hero 区域的背景图（加上暗色遮罩）
2. 把 mockup.png 放在"项目展示"区域
3. 把头像图片放在"关于我"区域
4. 所有图片使用 next/image 组件，设置合适的 width/height
5. 添加图片加载的 blur placeholder 效果
6. 确保图片做了 WebP 格式优化
```

### 第六步：生成 Favicon 和 OG Image

```text
帮我用 AI 生成的素材完善网站品牌：

1. 用 Ideogram 生成一个简约的 Logo（我的项目叫 [你的项目名]）
2. 把 Logo 转成 favicon（16x16, 32x32, 180x180）
3. 生成一张 OG Image（1200x630），用于社交媒体分享时的预览图
4. 把这些都配置到 Next.js 的 metadata 中
```

## Prompt 技巧

写好 AI 生图 prompt 的关键要素：

**主体** + **风格** + **色调** + **构图** + **质量词**

例如：
- ❌ "一个网站背景" （太模糊）
- ✅ "A minimalist abstract background with flowing gradient waves, dark navy to purple color scheme, subtle particle effects, 4K resolution, suitable for a SaaS landing page"

**常用质量词**：high quality, 4K, professional, clean, modern, minimal

## 通关条件

截图展示你的网站中至少使用了 2 张 AI 生成的图片素材（Hero 背景 + 至少一张其他用途的图）。

## 卡住了？

**Q: AI 生成的图片质量不好怎么办？**
多试几次，调整 prompt。加入更具体的描述词，或者参考其他人的 prompt（去 Civitai、PromptHero 找灵感）。

**Q: 图片太大影响加载速度？**
用 `next/image` 组件会自动优化。也可以用 TinyPNG 或 Squoosh 手动压缩。

**Q: 生成的图片有版权问题吗？**
大部分 AI 生图工具生成的图片归你所有，可商用。但建议查看各工具的具体条款。

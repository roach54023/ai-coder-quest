每个程序员的第一步都叫 "Hello World"。但传统的 Hello World 需要你自己敲代码。而你的 Hello World，只需要说一句话。

## 操作步骤

1. 在终端中进入一个你想放项目的文件夹（比如桌面）：

```bash
cd ~/Desktop
mkdir my-site
cd my-site
```

2. 启动你的 AI 搭档：

```bash
claude
```

3. 对 AI 说：

```text
在当前目录创建一个 index.html 文件，做一个全屏的欢迎页面：
- 正中间大字写 "Hello World"
- 下面一行小字写 "我的第一个网页"
- 深色背景，白色文字
- 字体要好看，居中显示
```

4. AI 会在当前目录生成一个 `index.html` 文件
5. 打开文件管理器（Mac: Finder / Windows: 资源管理器），找到这个文件，**双击打开**
6. 你会看到浏览器显示了你的页面！

## 卡住了？

- 如果看到的是代码文字而不是网页，右键文件 → 打开方式 → 选择浏览器（Chrome/Safari/Edge）
- 如果 AI 没有创建文件，检查你是否在正确的目录（`pwd` 命令查看当前目录）

## 通关小贴士

> 你刚才做了什么？你说了一句话，一个网页就出现了。传统方式你需要学 HTML 语法、CSS 样式、文件结构——大约需要一周的课程。你花了 30 秒。

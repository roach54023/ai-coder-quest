Node.js 是一个运行环境——你后面用到的AI工具和开发工具都需要它。把它想象成一台发动机：你不需要理解它怎么工作，但没有它车跑不起来。

## 操作步骤

1. 打开浏览器，访问 https://nodejs.org
2. 点击左边的 **LTS**（长期支持版）按钮下载
3. 打开下载的安装包，**一路默认/下一步**（不需要改任何选项）
4. 安装完成后，**关闭终端再重新打开**（重要！）
5. 在终端输入以下命令后按回车：

```bash
node -v
```

6. 如果看到类似 `v22.x.x` 的输出，说明安装成功

## 通关条件

粘贴 `node -v` 的输出结果（如 `v22.11.0`）

## 卡住了？

- 如果输入 `node -v` 提示"command not found"，说明终端没有刷新。关掉终端重新打开再试。
- Windows 用户如果安装后仍不识别，重启电脑一次即可。

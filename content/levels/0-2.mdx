上一关你把 AI 助手装好了。这一关做两件事：**让 AI 帮你写出第一个网页**，然后**把代码保存到 GitHub**。

GitHub 是全球最大的代码托管平台，你以后所有的项目都会放在这里。现在就开始用，后面每一章都会用到。

---

## 第一部分：让 AI 写出第一个网页

### 第一步：新建项目文件夹

先在桌面上建一个文件夹来放你的项目。

**Mac：** 打开"访达"，在桌面右键 → 新建文件夹，命名为 `hello-world`

**Windows：** 在桌面右键 → 新建 → 文件夹，命名为 `hello-world`

然后在 AI 工具里选中这个文件夹作为工作目录（Codex 点左上角选文件夹，Trae 点"打开文件夹"）。

### 第二步：让 AI 帮你写一个网页

在对话框输入：

```text
帮我创建一个 index.html 文件，内容是一个好看的个人欢迎页面，显示"Hello World! 我的 AI 编程之旅开始了"，背景用渐变色，文字居中，有一点动画效果。
```

等 AI 写完后，它会在你的项目文件夹里生成一个 `index.html` 文件。

> 🎯 注意看 AI 是怎么工作的：它理解你的需求 → 写出代码 → 保存到文件。你全程不用写一行代码。

### 第三步：在浏览器里打开看效果

找到项目文件夹里的 `index.html`，双击用浏览器打开。

你应该能看到一个带渐变背景、文字居中、有动画效果的页面。如果不满意，可以继续跟 AI 说：

```text
把背景换成深色系的，加一个打字机动画效果
```

AI 会帮你修改，刷新浏览器就能看到新效果。

> ✨ 恭喜！你已经完成了第一次"说话→出活"的完整循环。

---

## 第二部分：把代码保存到 GitHub

你的代码现在只在你的电脑上。GitHub 是程序员的"云端硬盘"——把代码放上去，换电脑也不怕丢，还能和别人协作。

### 第四步：注册 GitHub 账号

打开 [github.com](https://github.com)，点右上角 **Sign up**，用邮箱注册一个免费账号。

> 💡 GitHub 账号是程序员的"身份证"，后面部署网站、展示作品都要用到，现在就注册好。

### 第五步：创建你的第一个仓库

登录后，点右上角的 **「+」** → **「New repository」**，填写：

- **Repository name：** `hello-world`（仓库名，就是项目名）
- **Description：** 我的第一个 AI 网页（可选）
- 选 **Public**（公开，别人能看到你的作品）
- **不要**勾选 "Add a README file"

点 **「Create repository」**，仓库创建成功！

### 第六步：让 AI 帮你把代码推送到 GitHub

回到 AI 对话，输入：

```text
帮我把这个项目初始化 git 仓库，并推送到 GitHub。
我的 GitHub 用户名是：[你的用户名]
仓库名是：hello-world
```

AI 会给你一系列 git 命令，按照它的指引在终端里执行。

> 💡 **什么是终端？** Mac 叫"Terminal"（在应用程序 → 实用工具里），Windows 叫"PowerShell"（在开始菜单搜索）。如果你用 Trae，它内置了终端，直接在里面执行就行。

执行完成后，刷新你的 GitHub 仓库页面，应该能看到 `index.html` 文件出现了。

网站已经上线了。这一关把它打磨成一个你**真正愿意发给别人看**的作品集——加入动画效果、完善响应式、加上联系方式。

做完之后，这就是你的第一个正式作品，可以放在简历里、发给潜在客户、或者发到社交媒体上。

## 操作步骤

### 第一步：加入入场动画

对 AI 说：

```text
帮我给作品集网站加入以下动画效果：

1. 首页 Hero 区域：主标题有打字机动画效果（逐字出现）
2. 关于页面：技能标签有一个从左滑入的动画，依次出现（stagger 效果）
3. 作品卡片：页面加载时从下往上淡入，每张卡片延迟 0.1 秒（stagger）
4. 页面切换：路由切换时有 fade in 过渡动画（约 300ms）

如果需要安装动画库，推荐用 framer-motion，帮我安装并配置好。
```

等 AI 完成后，逐一测试这几个动画效果，确认都正常运行。

### 第二步：完善交互功能

继续对 AI 说：

```text
继续完善以下功能：

5. 加一个"回到顶部"的浮动按钮（滚动超过一屏后才显示，点击平滑滚动到顶部）
6. 作品卡片点击后弹出详情模态框（显示项目描述、技术栈、链接）
7. 底部加一个简单的联系表单（姓名、邮箱、留言，纯前端，点击发送后显示"感谢你的留言！"）
8. 确保所有页面在手机上完美显示（用 F12 开发者工具检查）
```

逐一检查每个功能是否正常，有问题继续让 AI 修复。

### 第三步：填入真实内容

现在把占位内容换成你自己的真实信息：

对 AI 说：

```text
帮我把作品集里的占位内容替换成真实内容：

关于我：
- 名字：[你的名字]
- 职业/身份：[比如：产品经理 / 设计师 / 大学生]
- 一段自我介绍：[2-3 句话描述你是谁、你做什么]
- 技能标签：[比如：产品设计、数据分析、AI 工具、Figma]

作品展示（至少 3 个，可以是真实项目或这门课做的项目）：
- 作品 1：[名称、描述、用到的技术/工具]
- 作品 2：[名称、描述]
- 作品 3：[名称、描述]

联系方式：
- 邮箱：[你的邮箱]
- GitHub：[你的 GitHub 链接]
- 其他：[微信/LinkedIn/微博，可选]
```

### 第四步：推送并验证线上效果

内容填好后，对 AI 说：

```text
帮我 git add 所有修改，commit 信息写"feat: 完善作品集内容和动画效果"，然后 push 到 GitHub。
```

等 Vercel 自动部署完成（1-2 分钟），用手机打开你的 Vercel 链接，检查：

- 动画效果是否正常？
- 手机上布局是否好看？
- 真实内容是否都显示正确？

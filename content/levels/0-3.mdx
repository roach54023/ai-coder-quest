## Claude Code 安装教程

### 第一步：安装

打开终端，输入：

```bash
npm install -g @anthropic-ai/claude-code
```

如果报权限错误（Mac），加 `sudo`：

```bash
sudo npm install -g @anthropic-ai/claude-code
```

### 第二步：启动

```bash
claude
```

首次运行会打开浏览器让你登录 Anthropic 账号。

### 第三步：验证

登录成功后，试着对它说：

```text
你好！请做一下自我介绍，你能帮我做什么？
```

看到 AI 的回复就说明一切就绪了。

### 基础操作

- **启动**：在项目目录下输入 `claude` 即可开始对话
- **退出**：输入 `/exit` 或按 `Ctrl+C`
- **清除上下文**：输入 `/clear`
- **查看帮助**：输入 `/help`
- **多行输入**：按 `Shift+Enter` 换行，`Enter` 发送

Claude Code 会自动读取你当前目录下的代码，所以**一定要先 `cd` 到你的项目文件夹再启动它**。

## OpenAI Codex 安装教程

### 第一步：安装

```bash
npm install -g @openai/codex
```

### 第二步：启动

```bash
codex
```

首次运行会要求你输入 OpenAI API Key。

### 第三步：验证

登录后同样试着和它对话，确认能正常回复。

### 基础操作

- **启动**：`codex` 进入交互模式
- **退出**：`Ctrl+C`
- **全自动模式**：`codex --full-auto "你的需求"` 让它自动完成任务

- 如果登录页面打不开，确认你的网络能访问国外网站
- 如果 API 报错 `insufficient_quota`，说明免费额度用完了，需要充值

你的代码现在只在你的电脑上。把它推到 GitHub 上，一方面是备份，另一方面后面部署要用。

## 操作步骤

### 第一步：git init 并创建 .gitignore

打开 https://github.com 注册账号（如果还没有）。

然后对 AI 说：

```text
帮我把这个项目初始化 Git 仓库并准备好推送：

1. 初始化 git 仓库
2. 创建 .gitignore（忽略 node_modules、.env 等）
3. 创建一个 README.md，简单介绍这个项目
4. 做第一次 commit
```

等 AI 完成后，你的代码就有了版本记录，可以准备推送了。

### 第二步：创建 GitHub 仓库

按 AI 的指引在 GitHub 网站上创建一个新仓库：

1. 登录 https://github.com，点击右上角 "+" → "New repository"
2. 给仓库起个名字（如 `my-fullstack-app`）
3. 选择 Public（公开）或 Private（私有）
4. 不要勾选任何初始化选项（README、.gitignore 等），因为你本地已经有了
5. 点击 "Create repository"，复制页面上显示的仓库地址

### 第三步：推送代码到 GitHub

把 GitHub 仓库地址告诉 AI，让它帮你推送：

```text
帮我把本地代码推送到 GitHub 仓库：
仓库地址：[粘贴你的仓库地址，如 https://github.com/yourname/my-fullstack-app.git]
```

执行完推送命令后，刷新 GitHub 页面，确认代码已经上去了。截图 GitHub 仓库页面，准备提交通关。

## 通关条件

粘贴你的 GitHub 仓库地址（如 `https://github.com/yourname/my-fullstack-app`）

## 卡住了？

- 如果推送时要求输入密码，需要设置 SSH key 或使用 Personal Access Token
- 让 AI 帮你："帮我设置 GitHub SSH key"
